O QUE APRENDI: 

Realizei um laboratório prático de Cloud Security no Google Cloud utilizando o Security Command Center, Cloud Shell e serviços da plataforma. Ativei APIs necessárias, configurei recursos do ambiente, analisei findings de segurança e investiguei vulnerabilidades identificadas pelo SCC. Trabalhei na correção de exposição de serviços de rede, revisando regras de firewall e restringindo acesso às portas SSH (22) e RDP (3389). Também criei regras de silenciamento de alertas, gerenciei configurações de segurança e exportei descobertas em formato JSONL para o Cloud Storage para fins de auditoria. Durante o laboratório, utilizei comandos pelo Cloud Shell para auxiliar na configuração, validação e gerenciamento dos recursos.
